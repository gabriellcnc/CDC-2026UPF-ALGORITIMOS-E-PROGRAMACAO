Tags HTML

As tags HTML são elementos que definem a estrutura e o conteúdo de uma página web.

Algumas das tags HTML mais comuns incluem:

  • <html> - Define o documento HTML
  • <head> - Contém informações sobre o documento
  • <title> - Define o título do documento
  • <body> - Contém o conteúdo visível da página
  • <h1> a <h6> - Definem títulos e subtítulos
  • <p> - Define um parágrafo
  • <a> - Cria um link
  • <img> - Insere uma imagem
