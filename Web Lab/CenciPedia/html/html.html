HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas da web. Ele é usado para estruturar o conteúdo da página, como texto, imagens, links e outros elementos. O HTML é composto por uma série de elementos, cada um com uma função específica, que são usados para organizar e formatar o conteúdo da página.

O HTML possui uma série de tags que permitem a criação de diferentes elementos na página.
